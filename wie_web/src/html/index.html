wie

이게 뭐예요?

wie는 옛 한국 피처폰에서 돌아가던 WIPI / J2ME 게임을 웹 브라우저에서 그대로 실행해 주는 에뮬레이터입니다.

설치 없이 모바일과 PC 어디서나 사용할 수 있고, 모든 처리는 브라우저 안에서 이루어집니다. 업로드한 파일은 외부로 전송되지 않습니다.

파일 업로드

휴대폰에서 추출한 파일을 ZIP 파일로 압축해서 그대로 업로드하세요.

※ ZIP 파일 안에 JAR 파일이 포함되어 있어야 합니다.
